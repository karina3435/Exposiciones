HTML considera que los párrafos son bloques de texto. Los navegadores hacen lo posible para adaptar el contenido de los párrafos a la ventana, a menos que los atributos NOWRAP o NOBR estén explícitamente especificados. Dentro de un párrafo, todos los espacios, tabuladores y saltos de línea cuentan como un espacio simple.Para colocar textos en bloques, se utiliza el par de etiquetas . Esta etiqueta puede incluir cualquiera de los atributos mencionados anteriormente.Los saltos de línea (saltar de una línea a la siguiente) se crean usando la etiqueta .La etiqueta se utiliza para insertar una línea horizontal.

Estos atributos suelen utilizarse para eliminar el margen, para lo cual deben valer cero.
